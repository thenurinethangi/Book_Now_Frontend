## Overview
**Synema** is a full-stack movie ticket booking platform designed to manage multiple cinemas and provide users with a smooth, modern booking experience.  
This repository contains the **frontend application**, built with React and TypeScript, focusing on usability, performance, and responsive design.

Users can browse movies, explore cinemas, view showtimes, select seats, and complete bookings securely. The interface also supports advanced features such as AI-generated movie summaries, seat locking, and real-time availability updates.
